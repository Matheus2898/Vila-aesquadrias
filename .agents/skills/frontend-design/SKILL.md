# Skill: UX Design de Alto Impacto (Antigravity)

## 🎯 Objetivo

Capacitar o Antigravity a gerar interfaces visualmente atraentes, intuitivas e orientadas à experiência do usuário, aplicando princípios fundamentais de UX com base em boas práticas consolidadas ao longo de 20 anos de experiência profissional.

---

## 🧩 Princípios Fundamentais de UX

### 1. 🎯 Ênfase / Dominância

**Objetivo:** Criar um ponto focal claro.

**Como aplicar:**

- Use cor, tamanho e posição para destacar o elemento principal.
- Limite a competição visual — apenas um elemento deve dominar.
- Utilize contrastes fortes para chamar atenção.

**Heurística prática:**

> “Se tudo chama atenção, nada chama atenção.”

---

### 2. 🔁 Unidade / Ritmo

**Objetivo:** Criar consistência e harmonia visual.

**Como aplicar:**

- Repita padrões de espaçamento, cores e tipografia.
- Use grids e alinhamentos consistentes.
- Estabeleça cadência visual (ex: listas, cards, linhas).

**Heurística prática:**

> “Interfaces previsíveis geram confiança.”

---

### 3. 🧭 Hierarquia

**Objetivo:** Guiar o olhar do usuário.

**Como aplicar:**

- Defina níveis claros (título, subtítulo, corpo).
- Use escala tipográfica consistente.
- Aplique contraste e espaçamento para separar níveis.

**Heurística prática:**

> “O usuário deve entender a tela em 3 segundos.”

---

### 4. ⚖️ Balanceamento

**Objetivo:** Distribuir peso visual de forma equilibrada.

**Como aplicar:**

- Use simetria ou assimetria intencional.
- Evite áreas “pesadas” demais em um lado.
- Distribua elementos com respiro adequado.

**Heurística prática:**

> “Equilíbrio visual reduz esforço cognitivo.”

---

### 5. 📏 Proporção / Escala

**Objetivo:** Criar relação clara entre elementos.

**Como aplicar:**

- Use proporções consistentes (ex: 8pt grid).
- Destaque elementos importantes com maior escala.
- Mantenha proporção lógica entre componentes.

**Heurística prática:**

> “Tamanho comunica importância.”

---

### 6. ⚡ Contraste

**Objetivo:** Diferenciar elementos claramente.

**Como aplicar:**

- Contraste de cor (claro vs escuro).
- Contraste de tamanho (grande vs pequeno).
- Contraste de forma e tipografia.

**Heurística prática:**

> “Contraste não é decoração, é comunicação.”

---

### 7. 🧬 Similaridade

**Objetivo:** Agrupar elementos relacionados.

**Como aplicar:**

- Use cores, formas e estilos consistentes para elementos semelhantes.
- Agrupe conteúdos relacionados visualmente.
- Evite inconsistências em componentes repetidos.

**Heurística prática:**

> “Elementos parecidos são percebidos como pertencentes ao mesmo grupo.”

---

## 🧠 Regras Avançadas (Experiência de 20 anos)

### 🔹 Redução Cognitiva

- Remova tudo que não agrega valor.
- Prefira clareza à estética exagerada.

### 🔹 Escaneabilidade

- O usuário não lê, ele escaneia.
- Use bullets, espaçamento e blocos curtos.

### 🔹 Feedback Visual

- Sempre informe estado: hover, ativo, loading.
- Interfaces sem feedback geram frustração.

### 🔹 Consistência Sistêmica

- Crie e siga um design system.
- Evite reinventar padrões.

### 🔹 Prioridade de Conteúdo

- Conteúdo vem antes do visual.
- Design deve servir à informação.

---

## 🛠️ Aplicação no Antigravity

### Quando gerar interfaces

- Sempre aplicar **hierarquia + contraste primeiro**.
- Depois ajustar **balanceamento e ritmo**.
- Finalizar com **refinamento de proporção e similaridade**.

NEVER use generic AI-generated aesthetics like overused font families (Inter, Roboto, Arial, system fonts), cliched color schemes (particularly purple gradients on white backgrounds), predictable layouts and component patterns, and cookie-cutter design that lacks context-specific character.

Interpret creatively and make unexpected choices that feel genuinely designed for the context. No design should be the same. Vary between light and dark themes, different fonts, different aesthetics. NEVER converge on common choices (Space Grotesk, for example) across generations.

**IMPORTANT**: Match implementation complexity to the aesthetic vision. Maximalist designs need elaborate code with extensive animations and effects. Minimalist or refined designs need restraint, precision, and careful attention to spacing, typography, and subtle details. Elegance comes from executing the vision well.

Remember: Claude is capable of extraordinary creative work. Don't hold back, show what can truly be created when thinking outside the box and committing fully to a distinctive vision.
